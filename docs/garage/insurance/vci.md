# 第三者责任险

> 第三者责任险简称三责或者三者。全称：商业第三者责任保险。是指被保险人允许的合格驾驶员在使用被保险车辆过程中发生的意外事故，致使第三者遭受人身伤亡或财产的直接损失，依法应当由被保险人支付的赔偿金额，保险人会按照保险合同中的有关规定给予赔偿。

----
## 保险渠道

渠道不固定，费用波动大。不再公示，以群消息为准吧。
