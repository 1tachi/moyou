# Triple T CLUB 仨树机车
---

Triple T CLUB 仨树机车位于751动力广场ACE CAFE后栋，主营各类摩托车专业清洗、保养、定制改装等服务。

不办理会员卡洗车费用在七八十左右，办理会员卡之后35元/辆。
